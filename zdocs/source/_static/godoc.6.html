type CtxWaitGroup ¶

CtxWaitGroup is a WaitGroup where Wait will exit with an error if a context cancels. TimeoutWaitGroup cannot be re-used once it closes.

    // contains filtered or unexported fields

func NewCtxWaitGroup ¶ 

func NewCtxWaitGroup(ctx context.Context) *CtxWaitGroup

NewCtxWaitGroup creates a new CtxWaitGroup that will signalClose when it's internal counter hits 0 OR ctx is cancelled.

func (*CtxWaitGroup) Add ¶ 

func (group *CtxWaitGroup) Add(delta int) error

Add adds delta to the WaitGroup. Returns an error if the passed context has been cancelled. Panics id the internal counter drops below -1.

func (*CtxWaitGroup) Done ¶ 

func (group *CtxWaitGroup) Done() error

Done decrements the WaitGroup by 1. Returns an error is the context has been cancelled. Panics if the internal counter drops below -1.

func (*CtxWaitGroup) Wait ¶ 

func (group *CtxWaitGroup) Wait() error

Wait waits until the WaitGroup closes OR the ctx expires.
